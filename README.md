# Stock Crash Risk Prediction

Machine learning project for predicting future stock drawdown risk using U.S. blue-chip equities.

## Current Progress

* ✅ Repository initialized
* ✅ Day3 completed: data processing and crash label construction
* 🔄 Day4 upcoming: feature engineering

## Dataset

10 U.S. blue-chip stocks:
AAPL, AMZN, GOOGL, JNJ, JPM, META, MSFT, NVDA, PG, XOM

## Label Definition

Crash = 1 if future 20-day maximum drawdown exceeds 10%.

## Model Plan

* LightGBM
* Random Forest
* SHAP Explainability
